# Unsplash API Server

## Pre-requisites

1. Register on [Unsplash Developer Portal](https://unsplash.com/developers).
2. Create an application.

## How to start

1. Run `npm install`.
2. Create a .env file.
3. Add the following variables in the file.
    - `UNSPLASH_ACCESS_KEY`
    - `UNSPLASH_SECRET_KEY`
4. Assign your Unsplash application values on these variables.
